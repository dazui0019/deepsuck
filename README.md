# DeepSuck

智能飞机杯。目前的想法是通过IMU来获取各个轴向上的加速度和角速度, 根据这些数据识别出一次抽插动作。看了抽插时各个轴向上的数据波形, 还是有比较明显的特征的, 所以这个方案理论上可行。

![飞机杯](/images/抽插特征.png)

## 数据处理

由于手的原因, 在使用飞机杯时是不会完全直上直下的, 存在一定的角速度。然后目前看下来, 对于慢速的抽插动作角速度会更加敏感一点, 所以现在的难点就是需要把这些数据融合起来分析。

然后由于飞机杯的截面是圆的, 所以没有办法保证使用时, IMU能完全固朝向某一面。因此姿态解算也是必要的, 需要根据姿态来动态更新以那几个轴为主要的数据源, 因为不同的IMU朝向会导致运动分量被分配到不同的轴向上。所以还需要增加一个磁力计。
